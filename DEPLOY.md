# 🚀 Deploy no GitHub Pages - Guia Rápido

## Passos para publicar:

1. **Criar repositório no GitHub:**
   - Nome: `relatorio-abnt`
   - Público (Public)
   - Sem README inicial

2. **Upload dos arquivos:**
   - Vá para o repositório
   - "Add file" > "Upload files"
   - Arraste: index.html, README.md, .gitignore, package.json

3. **Ativar Pages:**
   - Settings > Pages
   - Source: "Deploy from a branch"
   - Branch: main
   - Save

4. **URL do site:**
   - `https://SEU_USERNAME.github.io/relatorio-abnt/`

5. **Configurar Google OAuth:**
   - Adicionar nas credenciais:
     - Origem: `https://SEU_USERNAME.github.io`
     - Redirect: `https://SEU_USERNAME.github.io/relatorio-abnt/`

## 🎯 Resultado:
Site online funcionando com autenticação Google completa!
